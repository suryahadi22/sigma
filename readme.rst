Ini adalah sebuah software yang digunakan untuk melakukan tracking dan pencarian pekerjaan.
Menggunakan sebuah event yang diatur sedemikian rupa

Lihat bagaimana ini mudah digunakan:

    Cukup unduh dan pasang di server anda.
    Ganti beberapa config di dalam dan sesuaikan dengan situs anda.

    Cukup mudah karena menggunakan kerangka CodeIgniter

Features
--------

- Responsive
- Load lebih cepat
